  1. 第一条指令@echo off，关闭回显
    第二叫指令D:，切换到博客站点目录所在的盘符，我这里是D盘
    第三条指令cd D:\Hexo\Hexo，进入站点目录，我这里是D:\Hexo\Hexo
    第四条指令git add -A commit -m 'test' git push推送到github
    第五条指令pause,暂停
