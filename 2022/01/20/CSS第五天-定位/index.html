CSS第五天-定位

静态定位：static

静态定位是默认值，写了边偏移也不会有效果的

相对定位：relative

  1. 相对定位根据自身原来的位置，进行定位

  2. 没有脱标，在页面中还占有位置 天生就是为了给绝对定位当爹用的

绝对定位：absolute

  1. 根据最近带有定位(非静态)的祖先元素来定位 子绝父相

  2. 脱离标准流，在页面中不占有位置

  3. 绝对定位，固定定位，浮动，可以直接设置宽度和高度，默认宽度是内容宽度

​ ! 父元素搭配子绝父相使用

固定定位：fixed

  1. 永远根据浏览器可视区域定位，不会随着滚动条的滚动而滚动

  2. 可以直接加宽高，默认宽度是内容宽度

  3. 脱离标准流，在页面中不占有位置

  4. 固定定位的案例 1. 解决分辨率问题有两种方法

定位层级：z-index

  1. 只有定位有这个属性，绝对定位、固定定位、相对定位

  2. 默认值：auto

  3. 可以设置正值或者负值，没有单位

水平方向居中：

文字，行内元素，行内块元素，给其父元素添加text-align: center;

块级元素 给当前元素设置 宽度和margin: 0 autp;

浮动元素 给其外面的标准流的父元素设置 宽度和margin：0 auto;

定位元素 left: 50% translateX(-50%);

垂直方向居中：

文字，行内元素 line-height 等于height

行内块 行高等于高 搭配vertical-align: middle;

块级元素，浮动元素 需要去测量

定位元素 top：50% 50% translateY(-50%);

行内元素和行内块元素 并列关系的 垂直居中对齐

设置vertical-align: middle;

常见问题：

1.文本框和表单按钮无法对齐

2.input和img无法对齐

  1. div中的文本框，文本框无法贴顶问题

  2. div不设高度由img标签撑开，此时img标签下面会存在额外间隙问题

  3. 使用line-height让img标签垂直居中问题

鼠标样式：

cursor: default; 小箭头

cursor: pointer; 小手 (较常用)

cursor: move; 移动

cursor: text; 文本

cursor: not-allowed; 禁止

隐藏元素：

visibility: hidden; 隐藏元素，隐藏后的元素还占有位置

display: none; 隐藏元素，隐藏后的元素不占有位置

溢出隐藏：

overflow: visible;

overflow: hidden;

overflow: scroll;

overflow: auto;

隐藏属性值：

默认值：overflow: visible;

溢出隐藏：overflow-y: hidden;

添加滚动条：overflow: scroll;

超出 添加滚动条：不超出不添加 overflow: auto;

精灵图：

合并成一张大图片，这张大图片称之为精灵图

减轻服务器的压力，提高页面加载速度

背景图片大小：

background-size：宽度 高度；

数字+px、百分比、contain、cover

只做了解：
  1. 表格边框合并
  2. 焦点选择器
  3. 链接伪类选择器
  4. 属性选择器
  5. 元素整体透明
  6. CSS画三角形
文字阴影:

属性名：text-shadow

给文字添加阴影效果，吸引用户注意

盒子阴影:

属性名：box-shadow

给盒子添加阴影效果，吸引用户注意，体现页面的制作细节

过渡:

属性名：transition

让元素的样式慢慢的变化，常配合hover使用，增强网页交互体验
